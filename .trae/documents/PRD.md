## 1. 产品概述

基于 Babylon.js 的实时协作白板式建筑平面图设计器，支持多人同时拖拽墙体、门窗、家具等建筑组件，所有操作通过 Yjs CRDT 同步至其他协作者，实现零冲突的协同设计体验。

- 目标用户：建筑师、室内设计师、装修规划师
- 核心价值：降低多人协作设计门槛，实时同步避免版本冲突，可视化平面图编辑提升设计效率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 设计师 | 输入房间号加入 | 创建/编辑/删除所有建筑组件，查看协作者光标 |
| 观察者 | 输入房间号加入（只读模式） | 查看平面图和协作者操作，不可编辑 |

### 2.2 功能模块

1. **设计画布页**：2D 平面图编辑画布、组件拖拽放置、实时协作光标显示、缩放平移
2. **属性面板**：选中物体的属性编辑（尺寸、位置、旋转、颜色等）
3. **工具条**：绘制工具（墙体、门窗、家具选择）、操作工具（选择、移动、删除、撤销重做）

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 设计画布页 | 场景渲染区 | 基于 Babylon.js 渲染 2D 俯视平面图，支持网格背景、缩放平移 |
| 设计画布页 | 协作者光标层 | 显示其他协作者的实时鼠标位置和选区高亮 |
| 设计画布页 | 物体拖拽交互 | 从组件库拖拽墙体/门窗/家具到画布，支持吸附网格和对齐 |
| 设计画布页 | 属性面板 | 选中物体后右侧弹出属性编辑面板，修改后实时同步 |
| 设计画布页 | 工具条 | 左侧工具栏：选择/墙体/门/窗/家具/删除/撤销/重做 |
| 设计画布页 | 协作者列表 | 顶部显示当前在线协作者头像和颜色标识 |
| 设计画布页 | 房间信息 | 显示房间号、连接状态、在线人数 |

## 3. 核心流程

用户打开应用 → 输入房间号加入 → 进入设计画布 → 从工具条选择组件类型 → 在画布上点击/拖拽放置 → 选中物体编辑属性 → 操作自动同步至其他协作者 → 其他协作者看到实时变化

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝灰 (#1a1d23) + 亮青色 (#00d4aa) 强调色，营造专业建筑设计软件的工业感
- 辅助色：暖橙 (#ff8c42) 用于选中/警告状态，浅灰 (#2a2d35) 用于面板背景
- 按钮风格：圆角矩形 (8px)，微透明玻璃质感背景
- 字体：JetBrains Mono（数值/尺寸显示）+ Noto Sans SC（中文界面）
- 布局风格：左侧工具条 + 中央画布 + 右侧属性面板，顶部协作信息栏
- 图标风格：线性图标 (Lucide)，2px 描边

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 设计画布页 | 场景渲染区 | 深色网格背景，亮青色网格线，白色墙体线条，缩放控件 |
| 设计画布页 | 协作者光标层 | 彩色圆点光标 + 用户名标签，跟随鼠标移动 |
| 设计画布页 | 工具条 | 竖向图标按钮列表，选中态亮青色高亮，悬停微发光效果 |
| 设计画布页 | 属性面板 | 滑块/输入框组合，标签+值两列布局，实时数值反馈 |
| 设计画布页 | 协作信息栏 | 用户头像色块列表，房间号显示，连接状态指示灯 |

### 4.3 响应式策略

- 桌面优先设计，最小支持 1280px 宽度
- 属性面板在窄屏时可折叠为抽屉
- 工具条始终固定在左侧

### 4.4 2D 场景指引

- 视角：正交俯视 2D 视图（Orthographic Camera）
- 网格：1m 间距主网格 + 0.1m 间距副网格
- 墙体：双线表示（内外墙皮），填充灰白色
- 门窗：墙体上的缺口 + 专用符号（门扇弧线、窗线）
- 家具：简化平面图符号（矩形+图标）
- 选中态：亮青色虚线边框 + 控制点
- 对齐辅助线：橙色虚线
